MLSimKit Tutorial - Surface Variable Prediction for Sample Dataset

Usage:
1. Install MLSimkit, this includes the sample dataset
2. Run the training pipeline: ./run-training-pipeline
3. Run prediction: ./run-prediction

To remove outputs and logs, run ./run-clean

You may run individual commands manually:

    cd <this directory>
    mlsimkit-learn --config training.yaml surface train

See User Guide for details.
