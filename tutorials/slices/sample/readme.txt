MLSimKit Tutorial - Slice Prediction for Sample Dataset

Usage:
1. Install MLSimkit, this includes the sample dataset
2. Run the training pipeline: ./run-train.sh
3. Run prediction: ./run-predict.sh

To remove outputs and logs, run ./run-clean.sh

You may run individual commands manually:

    cd <this directory>
    mlsimkit-learn --config config.yaml slices predict

See User Guide for details.
